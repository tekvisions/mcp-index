About The MCP Index

The MCP Index is a living, self-updating directory of open-source Model Context Protocol (MCP) servers — the connectors that give AI assistants like Claude access to databases, APIs, file systems and external services. Every server is ranked by momentum, recomputed daily from live GitHub signals, so the index surfaces what the MCP ecosystem is actually adopting right now. It is one of The Living Indexes, a fleet built and operated end-to-end by Kymata Labs' AI agents.

What is the MCP Index?

The MCP Index is a living directory of open-source Model Context Protocol servers — the connectors that give AI assistants access to databases, APIs, files and external tools. Each server is ranked by momentum and re-ranked every day from live GitHub signals, so the list reflects what the MCP ecosystem is adopting now. It is one of The Living Indexes by Kymata Labs.

How is momentum scored?

A 0–100 score blending log-scaled GitHub stars (55%), push-recency (32%, decaying to zero by about 180 days) and rising-newness (13%). A tool that shipped this week can outrank a larger tool that has gone quiet — momentum, not legacy.

How often is it updated?

Every day. A GitHub Action recomputes each tool's momentum from live GitHub signals and republishes automatically, with no human in the loop.

How it's made

The registry of record, made browsable.

The MCP Index pulls the official Model Context Protocol registry every day, dedupes to the latest version of each server, infers a category, and checks freshness from each server's last update. Nothing here is hand-maintained or fabricated — it's the registry's own data, made fast to search and pleasant to read, by the same agent stack that runs Kymata Labs.
